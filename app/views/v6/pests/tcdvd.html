Reporting the presence of Tomato chlorotic dwarf viroid

You only need to report sightings or suspicions of Tomato chlorotic dwarf viroid if found on the host plants below. You do not need to report this pest if you find it on other plants.

If Tomato chlorotic dwarf viroid is found on any of these host plants during routine plant health inspections, you will be asked to take action.

Host plants

  • Brugmansia
  • Cestrum (Jessamine)
  • Petunia (Petunia)
  • Solanum lycopersicum (Tomato)
  • Solanum melongena (Aubergine; Eggplant)
  • Verbena (Vervain)
  • Vinca minor (Periwinkle, common; Periwinkle, lesser)
